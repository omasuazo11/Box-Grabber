Final project for WPI's Unified Robotics 1: Mechanical Applications in Robotics course. 
The robot was tasked with grabbing 2 boxes and changing their heights without hitting a shelf, and then driving to a final goal through an obstacle course.
The code features 3 classes, one per main actuator in the robot, a robot class, and a class for each motor type. Each actuator class has a motor parameter, 2 of which are PWM servos, and the other is a DC motor.
